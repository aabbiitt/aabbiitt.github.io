The Real-Time Heart Rate Monitoring System is an IoT-based project designed to continuously monitor a user's heart rate and provide real-time data. This system utilizes sensors to capture heart rate information, which is then transmitted to a central platform where the data is analyzed and displayed. The system aims to enhance healthcare by offering instant insights into a user's heart health, enabling timely interventions and remote monitoring.

Online Exit Exam Website:

The Online Exit Exam Website is a web-based platform developed to facilitate the administration of exit exams for students. This platform provides a seamless user experience, allowing students to register, take exams, and receive results online. The system is designed to handle large volumes of users, ensuring secure and efficient exam processes. It includes features such as automated grading, result tracking, and comprehensive reporting for both students and administrators.
